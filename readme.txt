== Installation ==

1. Upload ultimate-robotstxt folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings-> Ultimate Robots.txt click on this menu, You will see Robots.txt editor.
4. Now You can easily edit the robots.txt file.
